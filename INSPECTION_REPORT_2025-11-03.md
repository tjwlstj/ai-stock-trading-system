## 3. 업데이트 내역

### 3.1 FastAPI 0.120.0 → 0.120.4 업데이트

이번 업데이트에서는 FastAPI를 0.120.0에서 0.120.4로 업그레이드하였습니다. 이는 보수적 패치 업데이트로, 호환성 문제 없이 버그 수정 및 안정성 개선을 포함합니다.

#### 업데이트 이유

FastAPI 0.120.1부터 0.120.4까지의 버전에서는 다음과 같은 중요한 버그 수정 및 개선사항이 포함되어 있습니다:

**FastAPI 0.120.1** (2025-10-27):
- Starlette 버전 범위 업데이트 (<0.50.0)
- 라이선스 정보 개선 (`pyproject.toml`에 `license` 및 `license-files` 추가)

**FastAPI 0.120.2** (2025-10-29):
- 🐛 **중요**: 중첩 모델 스키마 분리 버그 수정 (0.119.0에서 발생한 이슈 해결)
- 내부 의존성 업데이트 및 스폰서 추가

**FastAPI 0.120.3** (2025-10-30):
- ♻️ 의존성 내부 순환 재귀 개선 (2개 함수 상호 호출 → 1개 함수 자기 호출)
- ♻️ 의존성 내부 리팩토링 및 코드 단순화 (`get_param_sub_dependant` 제거)
- ♻️ 데이터클래스를 사용한 내부 구조 단순화
- 📝 문서 개선

**FastAPI 0.120.4** (2025-10-31):
- 🐛 **중요**: 최상위 앱에서 보안 스키마 추가 시 OpenAPI 생성 버그 수정

#### 업데이트 영향

- **호환성**: 완전 호환 (Breaking Changes 없음)
- **위험도**: 매우 낮음 (Low)
- **테스트 결과**: 모든 문법 검사 통과
