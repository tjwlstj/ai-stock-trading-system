### 0.125.0 (2025-12-17) - **Breaking Changes**

**리팩토링**:

**문서**:
- ⚰️ CI 및 소스 문서에서 Python 3.8 제거

**번역**:
- 🌐 포르투갈어, 스페인어, 독일어 번역 업데이트
- 🔥 베트남어, 페르시아어, 이모지 번역 제거
- 🔧 프랑스어 LLM 프롬프트 파일 추가

**내부 변경**:
- ⬆ `markdown-include-variants` 0.0.7 → 0.0.8
- 👷 커버리지 설정 업데이트, Pydantic 버전 커스터마이징 가능
- 👷 LLM 번역 CI 업데이트

**영향 평가**:

## 종합 분석

### Breaking Changes 요약

| 버전 | Breaking Change | 현재 프로젝트 영향 |
|------|-----------------|-------------------|
| 0.125.0 | Python 3.8 지원 중단 | **영향 없음** (Python 3.11 사용) |
| 0.126.0 | Pydantic v1 지원 중단 | **영향 없음** (Pydantic v2 사용) |
| 0.127.0 | Pydantic v1 사용 시 경고 | **영향 없음** (Pydantic v2 전용) |
| 0.128.0 | `pydantic.v1` 완전 제거 | **영향 없음** (`pydantic.v1` 미사용) |

### 주요 개선사항

1. **Traceback 개선** (0.124.0): 디버깅 경험 향상
2. **버그 수정**: 파라미터 별칭, tagged union, TYPE_CHECKING, 임의 타입 처리
3. **내부 리팩토링**: Python 3.9+ 구문, 성능 테스트, CI 개선
4. **문서 및 번역**: 다국어 지원 강화, LLM 기반 번역 시스템

### 업데이트 권장사항

**현재 버전 유지 권장** ✅

**이유**:
1. **호환성 문제 없음**: 모든 Breaking Changes가 현재 프로젝트에 영향 없음
2. **안정성 우선**: 0.124.4는 안정적으로 작동 중
3. **짧은 간격**: 이전 업데이트(12월 15일) 후 약 4주 경과
4. **새 기능 부재**: 0.125.0 ~ 0.128.0은 주로 Breaking Changes 및 내부 개선
5. **급한 보안 문제 없음**: 알려진 보안 취약점 없음
6. **빠른 릴리스 주기**: 짧은 기간(12월 6일 ~ 27일)에 9개 버전 릴리스로 안정성 검증 부족

**대안 (선택적)**:
- 테스트 환경에서 FastAPI 0.128.0 검증
- 베타 버전(v1.3.2-beta)으로 먼저 릴리스
- 최소 1-2주 모니터링 후 안정 버전 승격
